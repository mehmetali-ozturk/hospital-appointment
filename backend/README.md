# Hospital Website built with React

This project is a web-based hospital management system developed using React for the front-end and Springboot Framework for the back-end. It provides functionalities for managing patients, doctors, and appointments.

## Introduction

This hospital website aims to streamline the process of managing patient information, doctor schedules, and appointment bookings. It offers a user-friendly interface for both patients and hospital staff to interact with the system efficiently.

## Features

- **Patient Management**:
    - Register new patients
    - Update patient information
    - View patient records
- **Doctor Management**:
    - Add new doctors
    - Update doctor profiles
    - Manage doctor availability
- **Appointment Management**:
    - Book appointments
    - View and manage appointments
    - Send appointment reminders

## Technologies Used

- **Front-end**: React, Tailwind, HTML, CSS, JavaScript
- **Back-end**: Springboot Framework, CRUD API, Java
- **Database**: Mysql
- **Other**: JWT Bearer Token Authentication
